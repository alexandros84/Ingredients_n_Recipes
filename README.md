The exact specifications are the following (text in Greek):

Καλείσαι να αναπτύξεις μια web πλατφόρμα, μέσω της οποίας ο χρήστης θα είναι σε θέση να προσθέτει και να διαχειρίζεται συνταγές μαγειρικής, αποτελούμενες από συστατικά (τα οποία επίσης θα εισάγει και διαχειρίζεται ο χρήστης). Παράλληλα θα μπορεί ο χρήστης να διαχειρίζεται το αποθεματικό του από αυτά τα συστατικά. H εφαρμογή θα πρέπει στην αρχική της σελίδα να δείχνει σε λίστα τις συνταγές που μπορεί ο χρήστης να εκτελέσει, λαμβάνοντας υπόψιν το αποθεματικό, μαζί με ένα κουμπί "εκτέλεση συνταγής" το οποίο θα πρέπει να αφαιρεί από το αποθεματικό τις ποσότητες των συστατικών που χρησιμοποιήθηκαν. Τέλος, θα πρέπει να υπάρχει ένα API endpoint το οποίο σε json μορφή να δίνει το σύνολο των συνταγών.

Τεχνικά θα πρέπει να ακολουθηθούν οι παρακάτω οδηγίες:

Η πλατφόρμα θα πρέπει να στηθεί με τη χρήση Ruby on Rails
Η βάση δεδομένων που θα χρησιμοποιηθεί να είναι MySQL ή PostgreSQL
Η πρόσβαση στην πλατφόρμα να είναι authenticated και να υπάρχουν οθόνες registration και password reset
Κάθε χρήστης πρέπει να είναι σε θέση να δει και να διαχειριστεί μόνο τα δεδομένα που ο ίδιος έχει εισάγει στην πλατφόρμα
Ιδανικά να γίνει χρήση Twitter Bootstrap για το παρουσιαστικό
H γλώσσα της εφαρμογής είναι η Αγγλική.

Η ημερομηνία παράδοσης είναι 5/4(/17)

Thanks for reading!
